# 🚛 Frontend - Calculadora de Frete

Aplicação React que consome a API para calcular de frete.

🔌 Dependências principais

- React 19

- Vite

- Axios (para chamadas API)

## 🚀 Como rodar

```bash
  npm install
```
```bash
  npm run dev
```

## ⚙️ Variáveis de ambiente
Crie um .env baseado no .env.example.
